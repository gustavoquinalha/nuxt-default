# Nuxt default theme
> Nuxt.js project

## Build Setup

``` bash
# install dependencies
$ npm install # Or
$ yarn install

# serve with hot reload at localhost:3000/
$ npm run dev
$ Go to http://localhost:3000/

# build for production and launch server
$ npm run build
$ npm start

# generate static project and deploy
$ npm run generate
$ npm run deploy
```
